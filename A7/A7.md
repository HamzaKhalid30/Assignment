# A7: Discover Cryptography Used in Modern Networks

## Overview

While exploring modern network security, I identified that organizations use VPN (Virtual Private Network) protocols such as IPsec and TLS to secure communication across networks. These protocols use strong cryptographic techniques to protect data during transmission over untrusted networks like the internet.

## Practical

1. Open your system settings and connect to a VPN service (or observe any corporate VPN if available).
2. Check connection details (Network settings → VPN).
3. Use tools like:
   - `ipconfig /all` (Windows) or `ifconfig` (Linux)
   - Browser → search "what is my IP" to confirm VPN connection
4. Observe that your IP address changes and traffic is routed through a secure encrypted tunnel.
5. (Optional) Use online tools or VPN client logs to identify protocol (e.g., IPsec / OpenVPN / WireGuard).

## Findings and Details

- I discovered that modern networks use IPsec (Internet Protocol Security) to encrypt and authenticate data packets at the network layer.
- IPsec ensures confidentiality, integrity, and authentication by encrypting data and verifying communication between devices.
- VPNs use cryptographic protocols like IPsec, OpenVPN, and WireGuard to create secure tunnels over public networks.
- These tunnels protect sensitive data from interception, especially when using public or untrusted networks.
- Modern protocols use strong encryption algorithms such as:
  - AES-256 (symmetric encryption)
  - ChaCha20 (modern lightweight encryption)
  - RSA / ECC (key exchange mechanisms)
- The combination of these cryptographic techniques ensures secure communication in enterprise networks, remote access systems, and cloud environments.

## Reflection

This activity helped me understand how cryptography is implemented at the network level to secure communication. Technologies like VPN and IPsec play a critical role in protecting data across modern networks. It also showed that without encryption, data transmitted over networks can be easily intercepted, making cryptography essential for secure communication.
