## Findings and Details

### 1. Shodan
- A search engine for internet-connected devices.
- Helps identify exposed systems like cameras, servers, and IoT devices.

### 2. VirusTotal
- Allows users to upload files or URLs to check for malware using multiple antivirus engines.

### 3. Burp Suite (Online Testing Tool)
- Used for testing web application security and analyzing HTTP requests.
- Can interact with live web applications for vulnerability testing.

### 4. Google Dorking (Search-Based Security Technique)
- Uses advanced search queries to find vulnerable or exposed data online.
- Helps identify misconfigured systems and sensitive information.

### 5. Have I Been Pwned
- Checks if email addresses or passwords have been exposed in data breaches.
- Helps users take action to secure compromised accounts.

## Reflection

This activity helped me understand the importance of online security tools in identifying real-time threats. Tools like Shodan and VirusTotal provide valuable insights into vulnerabilities and risks present on the internet. It also showed that combining multiple tools can significantly improve security analysis and awareness.
