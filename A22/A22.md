# A22: Design and Implement a Privacy-Preserving Technique

## Overview

Privacy-preserving techniques are methods used to protect sensitive user data while still allowing systems to function. These techniques ensure that personal information is not exposed or misused.

## Practical

1. Open VS Code / IDLE / Terminal
2. Write code:

```python
import hashlib

password = "mypassword123"

hashed_password = hashlib.sha256(password.encode()).hexdigest()

print("Original Password:", password)
print("Hashed Password:", hashed_password)
```

3. Run: `python file.py`

## Findings and Details

**Technique Used: Password Hashing**

- Instead of storing passwords in plain text, passwords are converted into a hashed format.
- Hashing ensures that even if data is leaked, original passwords cannot be easily recovered.
- Key privacy principles applied:
  - Data protection
  - Confidentiality
  - Secure storage
