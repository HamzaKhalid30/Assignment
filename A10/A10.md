# A10: Discover 5 Unique Access Control Devices

## Overview

While exploring different security systems, I identified various access control devices used to restrict and manage entry to physical spaces. These devices ensure that only authorized individuals can access specific areas, improving overall security. Access control systems work based on authentication methods such as what you have (card), what you know (PIN), and who you are (biometric).

## Findings and Details

### 1. RFID Card Reader
- Uses radio frequency identification (RFID) to grant access when a card or tag is scanned.
- Commonly used in offices and campuses.
- Provides fast and contactless authentication.

### 2. Biometric Scanner (Fingerprint / Face Recognition)
- Uses unique biological traits such as fingerprints or facial recognition.
- Offers high security because biometric data cannot be easily shared or duplicated.

### 3. Keypad Access Control (PIN-based)
- Requires users to enter a PIN code to unlock doors.
- Simple and cost-effective but less secure if codes are shared.

### 4. Smart Lock (Mobile / App-based Access)
- Allows access using smartphones, Bluetooth, or Wi-Fi.
- Can be controlled remotely and provides activity logs.

### 5. Proximity Card / Key Fob System
- Similar to RFID but often used in short-range contactless systems.
- Widely used in corporate offices and parking systems for quick access.

## Reflection

This activity helped me understand that access control is implemented using multiple technologies depending on security requirements. Each device has its strengths and weaknesses, and organizations often combine multiple methods (e.g., card + PIN or biometric) to enhance security. It also showed that modern systems are moving towards smart and biometric-based access control for better protection.
