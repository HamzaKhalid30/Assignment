## Findings and Details

### 1. Darktrace
- Uses AI to detect unusual network behavior in real time.
- Identifies threats like insider attacks and zero-day exploits.

### 2. CrowdStrike Falcon
- AI-powered endpoint protection platform.
- Detects malware and suspicious activity using behavioral analysis.

### 3. IBM QRadar
- Uses AI for Security Information and Event Management (SIEM).
- Correlates logs and detects threats across networks.

### 4. Microsoft Defender for Endpoint
- Uses machine learning to detect and prevent cyber threats.
- Provides automated investigation and response.

### 5. Palo Alto Cortex XDR
- Uses AI to detect advanced threats across endpoints and networks.
- Combines multiple data sources for better threat visibility.

## Reflection

This activity helped me understand how AI is transforming cybersecurity. AI-based tools can detect threats faster and more accurately than traditional methods. It also showed that automation plays a key role in modern security operations.
