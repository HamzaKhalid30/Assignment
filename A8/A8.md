## Overview

While exploring online privacy techniques, I identified that many users and organizations use VPN (Virtual Private Network) as a primary method to protect their online identity and data. A VPN encrypts internet traffic and hides the user's IP address, ensuring privacy during browsing.

## Practical

1. Install or open any VPN application (e.g., free or trial VPN).
2. Connect to a VPN server (e.g., different country).
3. Open browser and search: `what is my IP`
4. Note your IP address before and after connecting to VPN.
5. Observe that your real IP is hidden and replaced with another IP.

## Findings and Details

- I discovered that VPN creates an encrypted tunnel between the user and the internet, protecting data from interception.
- It hides the real IP address, making it difficult for websites, ISPs, or attackers to track user activity.
- VPN ensures confidentiality and anonymity, especially when using public Wi-Fi networks.
- Online privacy techniques also include:
  - Encryption (HTTPS, TLS)
  - Blocking tracking cookies
  - Using private browsers
  - End-to-end encrypted communication
- Without these techniques, user data such as browsing history, location, and personal information can be tracked or exposed.

## Reflection

This activity helped me understand how online privacy is maintained using tools like VPN and encryption. It showed that without proper privacy techniques, user data can be easily tracked. Implementing simple tools like VPN can significantly improve online security and anonymity.
