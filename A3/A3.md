# A3: Discover Security Concepts Used in Your House

## Overview

In my home environment, I focused on analyzing the security of my Wi-Fi network. I found that my network uses WPA3 encryption, which is currently one of the most secure wireless security protocols.

## Practical

1. Open Wi-Fi settings on your phone/laptop
2. Connect to your home Wi-Fi
3. Open network details
4. Control Panel → Network & Internet
5. Network and Sharing Center
6. Click your Wi-Fi
7. Click Wireless Properties → Security tab

## Findings and Details

- My home Wi-Fi is configured using WPA3 (Wi-Fi Protected Access 3), which provides strong encryption and secure authentication.
- WPA3 protects data transmitted over the network, ensuring that even if someone intercepts the traffic, they cannot easily read or modify it.
- It introduces advanced security features such as improved encryption and protection against brute-force attacks.
- Compared to older protocols like WPA2 or WEP, WPA3 significantly reduces the risk of attacks such as password cracking and man-in-the-middle attacks.
- This ensures that communication between devices (mobile, laptop, router) remains confidential and secure.
- Additionally, modern home security practices recommend combining WPA3 with network segmentation and strong passwords for better protection.

## Reflection

This activity helped me understand the importance of using modern encryption standards like WPA3. Securing a home network is essential because it acts as the entry point for all connected devices. Strong encryption ensures that attackers cannot easily intercept or exploit network communications.
