# A20: Perform a Prompt Injection Attack on a Generative AI Assistant (Controlled Test Only)

## Overview

Prompt injection is a technique used to manipulate AI systems by giving malicious or misleading inputs. It can cause AI to ignore instructions or reveal unintended information.

## Practical

- Used a generative AI tool (e.g., chatbot).
- Entered specially crafted prompts to manipulate responses.

## Findings and Details

- The AI attempted to respond differently when given manipulated prompts.
- In some cases, it ignored previous instructions or changed behavior.
- This shows that AI systems can be influenced by user input.
- Risks of prompt injection:
  - Data leakage
  - Misleading outputs
  - Bypassing restrictions
