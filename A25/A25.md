# A25: Enhance the Cybersecurity at Your Home

## Overview

Enhancing home cybersecurity involves identifying weak points in the home network and devices, then applying practical measures to strengthen protection. This activity focuses on implementing multiple security improvements to reduce the risk of unauthorized access and data breaches.

## Practical

1. Open router settings by typing `192.168.1.1` in browser
2. Login with admin credentials
3. Navigate to Security / Wireless settings
4. Check and update the following:
   - Wi-Fi password strength
   - Encryption type (WPA3/WPA2)
   - Firmware version
5. Disable remote management if not needed
6. Enable firewall settings on router

## Findings and Details

- Checked the home router settings and identified several areas for improvement.
- Security enhancements applied:
  - Changed default router admin username and password
  - Updated Wi-Fi encryption from WPA2 to WPA3
  - Set a strong Wi-Fi password (12+ characters, mixed symbols)
  - Disabled WPS (Wi-Fi Protected Setup) as it is vulnerable to brute-force attacks
  - Enabled router firewall to block suspicious incoming traffic
  - Updated router firmware to the latest version to patch known vulnerabilities
  - Enabled MAC address filtering to allow only known devices
  - Created a separate guest network for visitors to isolate main network

- Additional device-level enhancements:
  - Enabled screen lock with strong PIN/biometric on all devices
  - Enabled automatic updates on all connected devices
  - Installed antivirus software on laptop and PC

## Reflection

This activity helped me understand that home cybersecurity requires active effort and regular maintenance. Simple steps like updating passwords, enabling firewalls, and keeping firmware up to date can significantly reduce the risk of attacks. It showed that even home networks can be vulnerable if not properly configured.
